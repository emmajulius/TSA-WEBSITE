# TSA Digital Solutions Website

A modern, high-quality static website for TSA (Technique, Speed and Accuracy) Digital Solutions built with React and Vite.

## Features

- **Modern Design**: Clean, professional UI with smooth animations
- **Fully Responsive**: Works seamlessly on all devices
- **Fast Performance**: Built with Vite for optimal loading speeds
- **Smooth Scrolling**: Enhanced navigation experience
- **Contact Form**: Interactive contact form with validation
- **Portfolio Showcase**: Filterable project portfolio
- **Service Highlights**: Comprehensive service offerings display

## Sections

1. **Hero Section**: Eye-catching landing area with company tagline
2. **About Section**: Company values (Technique, Speed, Accuracy) and mission
3. **Services Section**: Detailed service offerings
4. **Portfolio Section**: Showcase of projects with filtering
5. **Contact Section**: Contact form and information
6. **Footer**: Additional links and company information

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Customization

All colors and styling can be customized through CSS variables in `src/index.css`:

- `--primary-color`: Main brand color
- `--secondary-color`: Accent color
- `--text-dark`: Dark text color
- `--text-light`: Light text color
- `--bg-light`: Light background color
